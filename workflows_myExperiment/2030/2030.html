Descriptions (3)
Uses the parsed Uniprot results (see workflow http://www.myexperiment.org/workflows/26.html) to retrieve information from PubMed.
Comparison of the genome of <i>Bacillus anthracis</i> to closely related strains will allow the discovery of proteins which may be involved in pathogenicity. Based on the biochemical pathways where the protein interacts, vaccines can be designed for these unique proteins. By gaining insight into the biochemical pathways that the unique proteins are involved in, the proteins can also be assessed for potential quality as vaccines. The key components of the vaccine finding system are the following: The vaccine finding system reads in at least two annotated whole genome sequences in EMBL or GenBank format; the source (non-pathogen) and the target (<i>Bacillus anthracis</i>). At this point, the workflow runs a BlastP search to compare the two strains of bacteria. Assessment of the evolutionary relationship between the two genomes ascertains whether they are closely related enough to derive meaningful comparisons. The vaccine finding system then parses the proteins from the data set that are only coded for in the target genome <i>Bacillus anthracis</i>, giving a new data set of unique proteins which can then be assessed in the KEGG PATHWAY database. The position of the unique proteins in the pathways will give a good indication of the enzymatic role of the proteins in <i>Bacillus anthracis</i>. The vaccine finding system then records its findings in an HTML file, as well as additional information such as annotations from Gene Ontology, functional annotations and literature references.
The drug repurposing workflow system screens at least 20 bacterial proteomes against this set of proteins that are already being treated against using established drugs. By screening the bacterial proteomes it will be possible to find proteins of highly similar structure to those that are existing drug protein targets and so this will infer that it is highly likely that the drugs can be used as antimicrobials against these proteins of highly similar structure. Proteomes that will be screened belong to Gram positive bacteria, with special focus on <i>Staphylococcus aureus</i>. We look at a variety of species and strains as the company advised. A mixture of screening targets could potentially identify targets for broad-spectrum antimicrobial development. The drug repurposing workflow shows where in the KEGG pathways the proteins are active and therefore where the action of the drug will be in the biochemical pathways of the bacteria. The workflow will update the Ondex database where new information on Gene Ontology annotations, functional annotation and literature references are available. All information about each unique protein is made available in an HTML document.

deXMLer beanshell

Script

String input = in1;

String pat = "<[^>]+>[^<]+";

java.util.regex.Pattern p = java.util.regex.Pattern.compile(pat);
java.util.regex.Matcher matcher = p.matcher(input);

String output = "";
List list = new ArrayList();

while(matcher.find())
{
	String match = input.substring(matcher.start()+1, matcher.end());
	if (match.contains("abstractText") || match.contains("fullName")
		|| match.contains("title") || match.contains("dx.doi.org")
		|| match.contains("yearOfPublication") || match.contains("volume")
		|| match.contains("pageInfo"))
	{
		list.add(match);
	}
}
String authors = "";
String abstractText ="";
String year = "";
String volume ="";
String pageInfo ="";
String link ="";
String journal = "";
String title = "";

for (Object o: list)
{
	String word = o.toString();
	int length = word.length();
	if (word.contains("abstractText"))
	{
		abstractText = word.substring(13, length);
	}
	else if (word.contains("fullName"))
	{
		authors += word.substring(9) +", ";
	}
	else if (word.contains("dx.doi.org"))
	{
		link = word.substring(4);
	}
	else if (word.contains("yearOfPublication"))
	{
		year = word.substring(18);
	}
	else if (word.contains("volume"))
	{
		volume = word.substring(7);
	}
	else if (word.contains("pageInfo"))
	{
		pageInfo = word.substring(9);
	}
	else if (word.contains("title") && journal.equals(""))
	{
		journal = word.substring(6);
	}
	else if (word.contains("title"))
	{
		title = word.substring(6);
	}

}


output = authors.substring(0,authors.length()-2) + " ("+year+"). "+title+". "+journal+", "+volume+ " :" + pageInfo+ ". "+link;

out1 = output;
