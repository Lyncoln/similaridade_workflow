Multi-parameter sensitivity analysis (MPSA)

Created: 2011-07-14 08:50:26      Last updated: 2011-07-21 10:08:42
  • Download Workflow

Exemplary multi-parameter sensitivity analysis (MPSA) experiment using Tav4SB (http://bioputer.mimuw.edu.pl/tav4sb/) WS operations. In this version of MPSA, ODEs simulations are basis of analysis of SBML model parameters space.

BufferedReader getReader (String fileUrl) throws IOException {
		InputStreamReader reader;
		try {
			reader = new FileReader(fileUrl);
		}
		catch (FileNotFoundException e) {
			// try a real URL instead
			URL url = new URL(fileUrl);
			reader = new InputStreamReader (url.openStream());
		}
		return new BufferedReader(reader);
	}



StringBuffer sb = new StringBuffer(4000);

BufferedReader in = getReader(fileurl);
String str;
String lineEnding = System.getProperty("line.separator");

while ((str = in.readLine()) != null) {
	sb.append(str);
	sb.append(lineEnding);
}
in.close();
filecontents = sb.toString();
