Longitude with beanshell

Created: 2011-10-06 04:06:57      Last updated: 2011-10-06 04:29:35
  • Download Workflow

This workflow takes in a string value or list of values and pulls out longitude coordinates formatted as ###-##(N or S) using the following regular expression: \d\d\d-\d\d(E|W). The coordinates are processed and formatted to be in the form required by the web service used in my "Elevation plot" workflow.

This workflow primarily contains string manipulation. For many of the text and list services used, the beanshell script needed to be modified in order to produce the formatting I required. All modified services contain documentation.

This workflow is very similar to my "Latitude with Netbeans" workflow, which performs similar manipulations between latitude coordinate formats.
