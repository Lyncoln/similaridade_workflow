This workflow retrieve known SNP located on the human chromosome 8 between positions 148200 and 148800 then propose Primers and Restriction enzymes that can be used within the same region. The same workflow can be used for any other region in the Human genome.

User can run the workflow for another chromosomic region, for that he have to change the Chromosome number ("chromosome_name" value="8"), start ("start" value="148200") and end position ("end" value="148800")within the las 4 lines of the first Biomart query Input description :
