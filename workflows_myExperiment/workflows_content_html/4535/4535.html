ENM resolution mix - multiple runs version

Created: 2014-11-13 15:20:55      Last updated: 2015-05-12 05:28:09
  • Download Workflow

Multiple runs version of the workflow for comparing results of two kinds of ecological niche models: one using only low resolution layers and the other using a random mix of low and high resolution layers. Users select the study region and the environmental variables considered to be the main drivers of a virtual species niche. The workflow is all based on the ENM components, which use the openModeller Web Service (OMWS). After getting initial parameters from the user, the workflow generates a random virtual niche. A random threshold within a given prevalence range is then used to separate suitable from unsuitable areas, from where random training and testing points are sampled. Training points are used to generate the models and independent testing points are used for external tests measuring the AUC. All steps (except mask and environmental layer selection) are repeated a given number of times to check if one of the the two kinds of models performs significantly better than the other. A single-run version of this workflow including model projections is available here: http://www.myexperiment.org/workflows/4536
