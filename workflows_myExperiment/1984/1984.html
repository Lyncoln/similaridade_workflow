Drug Re-Purposing Workflow

Created: 2011-03-25 20:06:13      Last updated: 2011-04-01 12:40:56
  • Download Workflow

The drug repurposing workflow system screens at least 20 bacterial proteomes against this set of proteins that are already being treated against using established drugs. By screening the bacterial proteomes it will be possible to find proteins of highly similar structure to those that are existing drug protein targets and so this will infer that it is highly likely that the drugs can be used as antimicrobials against these proteins of highly similar structure. Proteomes that will be screened belong to Gram positive bacteria, with special focus on Staphylococcus aureus. We look at a variety of species and strains as the company advised. A mixture of screening targets could potentially identify targets for broad-spectrum antimicrobial development. The drug repurposing workflow shows where in the KEGG pathways the proteins are active and therefore where the action of the drug will be in the biochemical pathways of the bacteria. The workflow will update the Ondex database where new information on Gene Ontology annotations, functional annotation and literature references are available. All information about each unique protein is made available in an HTML document.

deXMLer beanshell

Script

String input = in1;

String pat = "<[^>]+>[^<]+";

java.util.regex.Pattern p = java.util.regex.Pattern.compile(pat);
java.util.regex.Matcher matcher = p.matcher(input);

String output = "";
List list = new ArrayList();

while(matcher.find())
{
	String match = input.substring(matcher.start()+1, matcher.end());
	if (match.contains("abstractText") || match.contains("fullName")
		|| match.contains("title") || match.contains("dx.doi.org")
		|| match.contains("yearOfPublication") || match.contains("volume")
		|| match.contains("pageInfo"))
	{
		list.add(match);
	}
}
String authors = "";
String abstractText ="";
String year = "";
String volume ="";
String pageInfo ="";
String link ="";
String journal = "";
String title = "";

for (Object o: list)
{
	String word = o.toString();
	int length = word.length();
	if (word.contains("abstractText"))
	{
		abstractText = word.substring(13, length);
	}
	else if (word.contains("fullName"))
	{
		authors += word.substring(9) +", ";
	}
	else if (word.contains("dx.doi.org"))
	{
		link = word.substring(4);
	}
	else if (word.contains("yearOfPublication"))
	{
		year = word.substring(18);
	}
	else if (word.contains("volume"))
	{
		volume = word.substring(7);
	}
	else if (word.contains("pageInfo"))
	{
		pageInfo = word.substring(9);
	}
	else if (word.contains("title") && journal.equals(""))
	{
		journal = word.substring(6);
	}
	else if (word.contains("title"))
	{
		title = word.substring(6);
	}

}


output = authors.substring(0,authors.length()-2) + " ("+year+"). "+title+". "+journal+", "+volume+ " :" + pageInfo+ ". "+link;

out1 = output;
